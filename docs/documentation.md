# EduAI: Техникалық Құжаттама

## 1. Жоба туралы жалпы мәлімет
**EduAI** — Next.js 15 негізінде жасалған, жасанды интеллект мүмкіндіктерін пайдаланатын білім беру платформасы. Жобаның басты мақсаты — мұғалімдердің жұмысын автоматтандыру және оқушыларға жекелендірілген оқу жоспарын ұсыну.

## 2. Технологиялық архитектура
Жоба **Serverless Architecture** негізінде құрылған:
*   **Frontend**: React (Client Components) + Tailwind CSS.
*   **Backend**: Next.js App Router (Server Actions & Route Handlers).
*   **Database**: PostgreSQL (Supabase) + Realtime subscriptions.
*   **AI Engine**: Alem AI (GPT-4 негізіндегі модельдер).

## 3. Деректер қорының құрылымы (Schema)
Деректер қоры Supabase-те орналасқан және келесі негізгі кестелерден тұрады:
*   **profiles**: Пайдаланушылардың профильдері (id, email, full_name, role, grade, xp).
*   **community_quizzes**: Мұғалімдер жасаған тесттер (id, title, questions, creator_id).
*   **test_sessions**: Нақты уақыттағы тест сессиялары (id, pin, test_id, is_active).
*   **session_participants**: Сессияға қатысушылар мен олардың нәтижелері (session_id, user_id, score).

## 4. Негізгі алгоритмдер
### 4.1. AI Тест Генерациясы
Пайдаланушы тақырып пен сұрақ санын енгізгенде, жүйе Alem AI-ға арнайы "Prompt" жібереді. AI жауабын JSON форматында қайтарады, ол автоматты түрде платформаның форматына айналады.

### 4.2. Нақты уақыттағы рейтинг (Real-time Ranking)
Supabase-тің `Realtime` мүмкіндігі арқылы `session_participants` кестесіндегі өзгерістер (ұпайлар) мұғалімнің экранында ешқандай бетті жаңартпай-ақ (no reload) көрініп тұрады.

## 5. Қауіпсіздік (Security)
Барлық деректер **Row Level Security (RLS)** саясатымен қорғалған. Пайдаланушылар тек өздеріне тиесілі немесе рұқсат етілген деректерді ғана көре алады.
