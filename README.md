# EduAI — Жасанды Интеллектке Негізделген Әлеуметтік Білім Беру Платформасы

**EduAI** — заманауи білім беру жүйесін цифрландыруға және оқыту процесін жекелендіруге арналған инновациялық платформа. Бұл жоба дипломдық жұмыс аясында әзірленген және мұғалімдер мен оқушылар арасындағы өзара әрекеттестікті жасанды интеллект көмегімен жеңілдетуді мақсат етеді.

## 🚀 Негізгі Мүмкіндіктер

### 👨‍🏫 Мұғалімдер үшін:
*   **AI Quiz Generator**: Тақырып пен сыныпты енгізу арқылы бірнеше секундта тест сұрақтарын (5-20 сұрақ) автоматты түрде генерациялау.
*   **Live Sessions**: QR код арқылы сыныпта нақты уақытта (real-time) тесттер ұйымдастыру.
*   **Нақты уақыттағы рейтинг**: Оқушылардың жауаптарын бақылап, рейтинг кестесін тікелей эфирде көру.
*   **Мұғалім Панелі**: Оқушылардың үлгерімін талдау және тесттерді басқару.

### 👨‍🎓 Оқушылар үшін:
*   **Жекелендірілген Дашборд**: Оқу прогресі, XP ұпайлары және AI-дан арнайы ұсыныстар.
*   **Қонақ режимі (Guest Mode)**: Тіркелмеген оқушылардың QR код арқылы есімін жазып, бірден тестке қосылу мүмкіндігі.
*   **AI Mentor Feedback**: Тест аяқталған соң AI-дан жеке кеңестер мен деңгейді анықтау (Бастапқы/Орташа/Жоғары).
*   **Gamification**: Оқу процесін қызықты ететін XP және Streak жүйесі.

## 🛠 Технологиялық Стек

*   **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
*   **Frontend**: React 19, Tailwind CSS (Modern Glassmorphism UI)
*   **Backend & Database**: [Supabase](https://supabase.com/) (PostgreSQL, Realtime, RLS Security)
*   **AI Integration**: [Alem AI API](https://alem.ai/) (Quiz generation & Personal Mentoring)
*   **Authentication**: Supabase Auth (Role-based access control)

## 📦 Орнату және іске қосу

1.  Репозиторийді көшіріп алыңыз:
    ```bash
    git clone https://github.com/BekbolatBolebay/EduAI.git
    ```
2.  Тәуелділіктерді орнатыңыз:
    ```bash
    npm install
    ```
3.  `.env.local` файлын жасап, келесі кілттерді енгізіңіз:
    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
    ALEM_AI_API_KEY=your_alem_ai_key
    ```
4.  Жобаны іске қосыңыз:
    ```bash
    npm run dev
    ```

## 🔒 Қауіпсіздік

Платформада **Row Level Security (RLS)** жүйесі іске қосылған. Бұл әрбір пайдаланушының (мұғалім, оқушы, админ) тек өзіне тиесілі деректерге ғана қол жеткізуін қамтамасыз етеді.
